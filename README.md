# FarmMart - Farming Items E-Commerce Website

A modern, responsive e-commerce website for selling farming supplies, tools, seeds, equipment, and fertilizers.

## Features

- 🛒 **Shopping Cart** - Add items, update quantities, remove products
- 📱 **Responsive Design** - Works on mobile, tablet, and desktop
- 🔍 **Product Filtering** - Filter by category (Tools, Seeds, Equipment, Fertilizers)
- 💳 **Checkout System** - Complete cart and order functionality
- 📬 **Contact Form** - Get in touch with customers
- 🎨 **Modern UI** - Clean, professional design with smooth animations

## Products Available

- **Tools**: Garden Hoe, Heavy Duty Shovel, Garden Rake
- **Seeds**: Tomato Seeds, Corn Seeds, Basil Herb Seeds
- **Equipment**: Irrigation System, Wheelbarrow, Greenhouse Kit
- **Fertilizers**: Organic Fertilizer, NPK Fertilizer, Vermicompost

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start shopping!
